Memory Game Instrukcja
Witamy w Memory Game!
  • 1. Gra oferuje 3 poziomy trudności
  • 2. Gra rozpocznie się w momęcie kliknięcia na dowolną kartę
  • 3. Aby ukończyć grę należy odkryć wszystkie pary kart
  • 4. Po przejściu pierwszego poziomu trudności gra automatycznie uruchomi drugi i kolejny poziom
  • Powodzenia!
