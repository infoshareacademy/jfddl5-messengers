Memory Game Instrukcja
Witamy w Memory Game!
  • 1. Gra oferuje 3 poziomy trudności
  • 2. Gra rozpocznie się w momencie kliknięcia na dowolną kartę
  • 3. Aby ukończyć grę należy odkryć wszystkie pary kart
  • 4. Po ukończeniu pierwszego poziomu trudności gra automatycznie uruchomi drugi i kolejny poziom
  • 5. Po ukończeniu gry podaj swoje imię aby zapisać Twój wynik
  • Powodzenia!
